🚀 Backend pentru Stripe real
3 pași simpli — 15 minute
PASUL 1 — Crează cont Stripe
1. Mergi la stripe.com/register
2. Completează datele companiei (RomFlora / Flora Exclusiv Concept SRL)
3. Activează contul pentru plăți reale (KYC)
4. Din Dashboard → Developers → API Keys → copiază ambele chei
PASUL 2 — Backend Node.js (gratuit pe Render.com)
Creează fișierul server.js:
const express = require('express');
const stripe = require('stripe')('sk_live_...');
const app = express();
app.use(express.json());
app.use((req,res,next)=>{
  res.header('Access-Control-Allow-Origin','*');
  res.header('Access-Control-Allow-Headers','*');
  next();
});
app.post('/create-payment-intent', async (req,res) => {
  const intent = await stripe.paymentIntents.create({
    amount: req.body.amount, // cents
    currency: 'eur',
    metadata: { order: req.body.orderId }
  });
  res.json({ clientSecret: intent.client_secret });
});
app.listen(3000);
PASUL 3 — Deploy pe Render.com (gratuit)
1. render.com → New Web Service
2. Conectează GitHub cu server.js
3. Environment: STRIPE_SECRET_KEY=sk_live_...
4. Deploy → primești URL: https://romflora-api.onrender.com
5. Pune URL-ul în câmpul de mai jos
